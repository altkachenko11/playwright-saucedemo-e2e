# Test Strategy

## Scope

This project covers end-to-end testing of the SauceDemo web application.

## Test Types

Smoke tests

- Login
- Inventory page loads
- Successful checkout

Regression tests

- Add product to cart

Negative tests

- Checkout validation

## Tools

Playwright
TypeScript
GitHub Actions

## CI

Tests run automatically on:

- Push to main branch
- Pull requests
- Scheduled daily run
